- **Programming Language:** Java (Core Java)

  - Classes and Objects
  - Encapsulation
  - Constructors
  - Method Interaction
  - Basic Exception Handling
  - Modular Design

---

##  Project Structure:---

##  Class Description

###  Account.java
- Stores account number, account holder name, and balance
- Provides methods for deposit and withdrawal operations

###  Transaction.java
- Represents individual transactions
- Stores transaction type, transaction amount, and balance after transaction

### Bank.java
- Acts as a service/controller class
- Performs operations on Account objects
- Maintains transaction history

### UseBank.java
- Contains the `main()` method
- Demonstrates creation of account and execution of banking operations
